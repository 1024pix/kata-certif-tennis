# Règles du tennis

- 2 equipes : A et B
- afficher les points : 
  - 0 love
  - 1 fifteen
  - 2 thirty
  - 3 fourty
  - >= 4 win (avec condition)
  - (3 - 3) deuce
  - all : egalité
ex: love - fifteen (0 - 1); love all (0 - 0)

- win si : 
  - 2 points d'equart pour gagner
  - >= 4
  - si un point d'equart avantage ex: 4 - 5

# Usage

```js
# Input
checkScore(3, 3)
# Output
return 'deuce'
```
